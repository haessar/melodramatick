Open-source cultural engagement platform

Melodramatick

A reusable Django-based template for publishing cultural event data and collecting lightweight audience engagement signals.

View the GitHub repo See current examples

What is it?

Melodramatick provides a reusable pattern for creating themed audience engagement sites for cultural domains such as opera, ballet, and classical music. Each deployment can ingest structured event or performance data, publish browsable records, and allow users to mark performances they have attended.

How it works

  1. Ingest structured cultural event data.
  2. Publish a themed engagement site.
  3. Allow users to record attended performances.
  4. Generate safe aggregate outputs for partners and pilots.

For developers

The source code is available on GitHub and can be adapted to create new themed deployments.
